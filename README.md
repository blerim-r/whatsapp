## Main features

* Story 
* New Status
* Camera/Screenshot
* All/Missed Calls 
* Calling page
* Conversation list (search/delete)
* Send message
* Settings page
* Change profile picture


## Example

Insert all divs that you want to make like folder in a relative div container

## Usage
```sh
folder("#yourDiv", {title: "Div Title", template: "windows|mac|linux"});
```
